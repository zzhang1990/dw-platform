# 调度配置说明

## DolphinScheduler 配置

### 环境配置

1. 创建环境：`dw-platform`
2. 配置 Python 环境：`conda activate dw-platform`
3. 设置工作目录：`/opt/dw-platform`

### 任务参数

```bash
# Python 任务
python ${project_path}/scripts/etl/etl.py --dt ${dt} --layer ${layer}

# Shell 任务
bash ${project_path}/shell/run_etl.sh ${dt} ${layer}
```

### 时间参数

| 参数 | 说明 | 示例 |
|------|------|------|
| `${dt}` | 业务日期 | 2024-01-01 |
| `$[yyyy-MM-dd-1]` | 昨天 | 系统计算 |
| `$[yyyy-MM-1]` | 上月 | 系统计算 |

## 调度依赖

### DAG 设计原则

```
数据同步 (ODS)
    │
    ├── DIM 构建
    │
    └── DWD 清洗
           │
           └── DWS 汇总
                  │
                  └── ADS 计算
                         │
                         └── BI 刷新
```

### 任务依赖图

```
[ODS 同步任务]
      │
      ├── [DIM 维度任务]
      │         │
      │         └──────────┐
      │                    │
      └── [DWD 明细任务] ───┼── [DWS 汇总任务]
                           │         │
                           │         └── [ADS 指标任务]
                           │                    │
                           └────────────────────┘
```

---

## 调度周期

| 任务类型 | 调度周期 | 说明 |
|----------|----------|------|
| ODS 同步 | T+1 凌晨 | 每日凌晨执行前一天数据 |
| DIM 构建 | 日 | 全量覆盖 |
| DWD 清洗 | 日 | 增量处理 |
| DWS 汇总 | 日 | 滚动更新 |
| ADS 计算 | 日 | 全量/增量 |

### 执行时间建议

| 时间 | 任务 |
|------|------|
| 01:00 | ODS 同步开始 |
| 02:00 | DIM 构建 |
| 03:00 | DWD 清洗 |
| 04:00 | DWS 汇总 |
| 05:00 | ADS 计算 |
| 06:00 | BI 数据刷新 |

## 告警配置

### 告警规则

1. **任务失败**：立即告警
2. **任务超时**：超过预期时间 2 倍告警
3. **数据质量**：空值率/重复率超阈值告警

### 告警渠道

- 钉钉机器人
- 邮件通知
- 短信（重要任务）

## 补数据流程

### 单日补数据

```bash
# Shell 方式
bash shell/run_etl.sh 2024-01-15 all

# Python 方式
python scripts/etl/etl.py --dt 2024-01-15 --layer all
```

### 批量补数据

1. DolphinScheduler 补数据功能
2. 选择任务和日期范围
3. 按依赖顺序执行

### 补数据优先级

1. 先停止当天正常调度
2. 补数据完成后恢复
3. 确认数据完整性
