# 数据流向图

## 整体流向

```
源系统
  │
  ├── PostgreSQL ── CloudCanal(实时CDC) ──┐
  │                                       │
  ├── MySQL ────── DataX(批量) ───────────┼──→ ODS
  │                                       │
  ├── API ──────── Python脚本 ────────────┘
  │
  ▼
ODS (贴源层)
  │
  ├──→ DIM (维度层)
  │
  └──→ DWD (明细层)
         │
         └──→ DWS (汇总层)
                │
                └──→ ADS (应用层)
                       │
                       ▼
                    永洪BI
```

## 层级依赖关系

```
ODS ─────────────────────────────────────────┐
  │                                          │
  │    ┌── dim_user (用户维度)               │
  ├────┤                                     │
  │    └── dim_product (产品维度)            │
  │                                           │
  └──→ DWD                                   DIM
         │                                    │
         └──→ DWS                             │
                │                             │
                └──→ ADS ←────────────────────┘
                       │
                       ▼
                    永洪BI
```

## 任务调度依赖

```
[ODS 同步任务]
      │
      ▼
[DIM 维度构建] ←── 维度表数据准备
      │
      ▼
[DWD 明细清洗]
      │
      ▼
[DWS 汇总计算]
      │
      ▼
[ADS 指标计算]
      │
      ▼
[BI 报表刷新]
```
