# 技术选型说明

## 核心组件

### StarRocks

**选型理由：**
- MPP 架构，高性能 OLAP 查询
- 支持实时更新和批量导入
- 兼容 MySQL 协议，学习成本低
- 支持物化视图，加速查询

**版本：** 3.x

### DolphinScheduler

**选型理由：**
- 可视化 DAG 配置
- 支持多种任务类型（Shell、Python、SQL）
- 高可用架构
- 强大的任务依赖管理

**版本：** 3.x

### CloudCanal

**选型理由：**
- 支持 PostgreSQL CDC 实时同步
- 可视化配置，运维简单
- 数据一致性保障

### DataX

**选型理由：**
- 阿里开源，稳定性好
- 支持多种数据源
- 流式传输，内存占用低

### 永洪BI

**选型理由：**
- 国产 BI 工具，本土化支持好
- 性能优秀，支持大数据量
- 可视化能力强
- 支持自助分析

### Python

**选型理由：**
- 丰富的数据处理库
- 开发效率高
- 团队熟悉度高

**版本：** 3.12
