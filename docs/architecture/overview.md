# DW Platform 架构概述

```
┌─────────────────────────────────────────────────────────────────┐
│                         数据源层                                 │

└─────────────────────────────────────────────────────────────────┘
                              │
              ┌───────────────┼───────────────┐
              ▼               ▼               ▼
        ┌─────────┐     ┌─────────┐     ┌─────────┐
        │CloudCanal│     │ DataX   │     │ Python  │
        │ (实时)   │     │ (批量)  │     │ (API)   │
        └────┬────┘     └────┬────┘     └────┬────┘
             │               │               │
             └───────────────┼───────────────┘
                             ▼
┌─────────────────────────────────────────────────────────────────┐
```

## 调度架构

```
┌─────────────────────────────────────────────────┐
│              DolphinScheduler                    │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐      │
│  │ 数据同步  │  │ ETL 任务 │  │ API 采集 │      │
│  │   任务    │  │   调度   │  │   任务   │      │
│  └──────────┘  └──────────┘  └──────────┘      │
└─────────────────────────────────────────────────┘
```

## 数据流向

1. **数据采集**: 源系统 → ODS 层
2. **数据处理**: ODS → DIM/DWD → DWS → ADS
3. **数据应用**: ADS → 永洪BI → 报表/仪表盘
